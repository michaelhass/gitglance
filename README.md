# gitglance

Simple git terminal UI written in Go using [Bubble Tea](https://github.com/charmbracelet/bubbletea).
![gitglance demo](assets/gitglance_demo.gif)

## Features
- Stage files ✔️
- Unstage files ✔️
- Reset files ✔️
- View diffs ✔️
- Commit ✔️

## Installation
### Go
```
go install github.com/michaelhass/gitglance@latest
```
